Práctica #1: Manejo de entradas y salidas digitales

Duración: 2 Semanas.

Objetivo General:

  • Desarrollar programas en lenguaje ensamblador para el manejo de entradas y salidas digitales de un microcontrolador orientados a la resolución de problemas
Objetivos Específicos:
  • Elaborar diagramas de flujo, diagramas de estado, seudocódigos y otras herramientas para describir la solución de problemas mediante el uso de un microcontrolador
  • Escribir programas en lenguaje ensamblador utilizando el entorno AVR-Studio 7
  • Simular el funcionamiento de un circuito digital basado en el microcontrolador ATMEGA 328p
  • Verificar la operación utilizando el hardware disponible en el laboratorio

Elabore un diagrama de flujo o seudocódigo que describa un programa para la siguiente aplicación:

Se requiere elaborar un sistema automatizado para la elaboración de cloro partiendo de la materia prima NaClO, además del envasado del mismo por medio de una línea de llenado conformado por los elementos mostrados en la figura 1.

Figura 1: Componentes del sistema automático para la elaboración de cloro

Los sensores que miden el nivel de líquido dentro de los tanques A y B trabajan de la siguiente manera:

  • LLA/LLB: Nivel bajo del tanque A/B. LLA/B = 1 => fluido por debajo del sensor. LLA/B = 0 => fluido a nivel o por encima del sensor.
  • HLA/HLB: Nivel alto del tanque A/B. HL = 1 => fluido a nivel o por encima del sensor. HL = 0 => fluido por debajo del sensor

Los sensores de llenado y de parada para el envasado y movilidad de la cinta transportadora funcionan de la siguiente manera:

  • SLL: Sensor de llenado para los envases. SLL = 1 => Envase lleno. SLL = 0 => envase sin llenar.
  • SP: Sensor de parada. SP = 1 => Se detectó un envase en la posición adecuada. SP = 0 => No se ha detectado presencia del envase

El Sensor de alta presión funcionan de la siguiente manera:

  • PHS: Sensor de alta presión. PHS = 1 => Alta presión. PHS = 0 => Presión adecuada.

Considere para el resto de los componentes lo siguiente:

  • Válvulas: Vx = 1 => Válvula abierta. Vx = 0 => Válvula cerrada.
  • Bombas: Bx = 1 => Bomba encendida. Bx = 0 => Bomba apagada.
  • Motores: Mx = 1 => Motor encendido. Mx = 0 => Motor apagado.
FUNCIONAMIENTO:
  • Inicialmente los tanques A y B se encuentran vacíos.
  • El proceso inicia cuando se presiona el pulsador START.
  • Una vez iniciado el proceso, se abre la válvula V1 para permitir el paso de agua hacia el tanque A.
  • Cuando el agua llegue al máximo nivel (Impuesto por HLA) se cerrara la válvula V1 impidiendo que el tanque se desborde
  • Una vez que el tanque A se encuentre lleno, se procede al mezclado para la preparación del cloro, por lo que el tanque B se llenara con un 87.5% de agua en el cual, al llegar a este nivel, se encenderá el motor M1 para dar inicio al proceso de mezclado mientras que el 12.5% restante se va llenando con la materia prima NaClO, asuma que el tanque de la materia prima no posee restricciones.
  • Al llenarse por completo el tanque B se apaga el motor M1 por lo que el proceso de mezclado se detiene y luego se abre la válvula V4
  • Para que inicie el proceso de llenado de envases, se debe verificar que haya un envase vacío en la línea de llenado, para esto haga uso de los sensores SLL y SP
  • Una vez detectado dicho envase, se abre la válvula V5 y simultáneamente se enciende la bomba BB, este proceso ocurre hasta que el envase se encuentre totalmente lleno a lo que, se apaga la bomba BB y se cierra la válvula V5
  • Una vez ocurrido esto, se enciende el motor M2 para mover la cinta transportadora que permitirá el ingreso de un nuevo envase vacío a lo que, al ser detectado por SP el motor M2 se apaga deteniendo el desplazamiento.
  • Este proceso se repite tantas veces lo permita el nivel del tanque B, por lo que, al vaciarse por completo se cierra la válvula V4 e inicia un nuevo proceso de mezclado.
CONSIDERACIONES:
  • El sensor PHS es utilizado para protección de la bomba del tanque A, si PHS detecta un nivel alto de presión la bomba A se apagará
  • El sensor de llenado (SLL) detecta cuando el envase está completamente lleno.
  • El sensor de parada (SP) detecta la posición del envase cuando éste se encuentra justo debajo de la línea de llenado deteniendo, en este sentido, al motor M2
  • Considere la necesidad de añadir sensores si este los requiere.
  • Tome en cuenta la simulación de los motores a través de LED’s, dado que, el diagrama de potencia no será objetivo de estudio en este apartado
  • El sistema arranca con una señal activo bajo
CONEXIÓN DE TERMINALES DE ENTRADA / SALIDA
Para la implementación del algoritmo elaborado anteriormente deberá considerar lo siguiente:
  • El pulsador START estará conectado al terminal PC0
  • El sensor HLA se representará mediante un interruptor conectado en el terminal PC1
  • El sensor LLA se representará mediante un interruptor conectado en el terminal PC2
  • El sensor HLB se representará mediante un interruptor conectado en el terminal PC3
  • El sensor LLB se representará mediante un interruptor conectado en el terminal PC4
  • El sensor PHS se representará mediante un interruptor conectado en el terminal PB1
  • El sensor SP se representará mediante un interruptor conectado en el terminal PB3
  • El sensor SLL se representará mediante un interruptor conectado en el terminal PB4
  • La bomba BA se representará mediante un led conectado en el pin PB0
  • La bomba BB se representará mediante un led conectado en el pin PD7
  • El motor M1 se representará mediante un led conectado en el pin PD6
  • El motor M2 se representará mediante un led conectado en el pin PD5
  • Las Válvulas de V1 hasta V4 se representarán mediante led's conectados en los pines desde PD4 hasta PD1
  • La válvula V5 se representará mediante un led conectado en el pin PD0
